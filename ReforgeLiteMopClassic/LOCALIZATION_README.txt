# ReforgeLiteMopClassic

This addon now supports multilingual localization with automatic language detection.

Supported locales:
- English (enUS)
- Portuguese Brazilian (ptBR)
- Spanish (esES)
- German (deDE)
- French (frFR)
- Russian (ruRU)
- Korean (koKR)
- Simplified Chinese (zhCN)
- Traditional Chinese (zhTW)

Unknown locales automatically fallback to English.
